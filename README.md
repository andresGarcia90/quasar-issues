## 🚀 Quick Start

### Install the dependencies

```bash
yarn
# or
npm install
```

### Set Environment variables in .env file

```
GITHUB_TOKEN=github_token
GITHUB_USER=facebook
GITHUB_REPO=react


```

### Start the app in development mode (hot-code reloading, error reporting, etc.)

```bash
quasar dev
```
